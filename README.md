# ToDo List App — Full-Stack

A full-stack task management app with Google OAuth authentication.

## Stack
- **Front-end:** React, TypeScript
- **Back-end:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** Google OAuth 2.0

## Features
- Secure login via Google OAuth 2.0
- Create, edit, delete and complete tasks
- Responsive design
- RESTful API

## How to Run
\`\`\`bash
git clone https://github.com/Nivetha07-dev/todo-list-react-node
cd todo-list-react-node
\`\`\`

**Backend**
\`\`\`bash
cd server
npm install
npm start
\`\`\`

**Frontend**
\`\`\`bash
cd to-do-list
npm install
npm start
\`\`\`

Requires a MongoDB connection string and Google OAuth credentials in a `.env` file — see `.env.example`.
